Função de densidade de probabilidade da distribuição beta:

Observação: a beta com parâmetros a e b iguais a 1 é uma distribuição uniforme!

\(\color{red}{\text{A f.d.p. da distribuição beta é:}}\)

\[f(x| \alpha, \beta) = \frac{\theta^{\alpha - 1} (1 - \theta)^{\beta - 1}}{\beta(\alpha, \beta)} . I_{\{0 \le x \le 1\}}\]

########### Distribuição Beta (f.d.p.) ################
fdp_logbeta <- function(x, alpha, beta){
  lik <- ((alpha - 1) * log(x) + (beta - 1) * log(1 - x)) - log(beta(a = alpha, b = beta))
  return(lik)
}

Implementando uma função de verossimilhança:

\(\color{red}{\text{A f.d.p. da distribuição binomial é:}}\)

\[f(x| n, \theta) = \binom{n}{x} . \theta^x . (1 - \theta)^{(n - x)} . I_{\{x = 0, 1, 2, ..., n\}}\] …Sendo n = número de lançamentos (N) e x = número de resultados desejados (z)! ***

\(\color{red}{\text{Entretanto, vamos desprezar o termo de combinação na função...}}\)

########### Distribuição binomial (f.d.p.) ################
fdp_bino <- function(x, n, theta){
  lik <- x * log(theta) + (n - x) * log(1 - theta)
  return(lik)
}

Estimando a verossimilhança dos lançamentos:

# Calculando a verossimilhança do dado para cada valor de theta:
pDadosTheta = fdp_bino(x = z, n = N, theta = theta)

Finalmente, vamos calcular a probabilidade da moeda ser justa, dado os resultados obtidos… Vamos calcular as probabilidades posteriores!

# Calculando o log dos posteriores de cada valor de theta:
pData <- sum(pTheta + pDadosTheta)
pThetaDados <- (pTheta + pDadosTheta) - pData
